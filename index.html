    Arkanoidx.

    Arkanoidx is a 2D game developed with libgdx game engine and Java language. It's a clone of classical Arkanoid created to teach about developing games with libgdx and Java

    Also, I have uploaded this project at GitHub to teach about code repositories and to share code about several developers who work together.
